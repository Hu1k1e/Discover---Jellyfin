API Settings

Your API key from themoviedb.org. Required for Upcoming and Recommended sections.
Full URL of your Jellyseerr instance (no trailing slash needed).
Found in Jellyseerr → Settings → General → API Key.

Stream Settings

The TMDB movie ID will be appended to this URL when "Stream Directly" is clicked. Example: https://stream.hulksmash.ca/movie/

Navigation Placement

Choose where the Discover navigation link appears. Sidebar adds it to the left drawer. Header adds it as a tab in the top navigation bar. A browser refresh is required after changing this setting.

Section Visibility

Displays movies you have marked as Liked in Jellyfin (KefinTweaks Watchlist compatible).
